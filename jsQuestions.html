4.) NaN, Infinity, Null and Undefined with proper explanation and examples:

    NaN:

  • In JavaScript, NaN is short for "Not-a-Number".
  • In JavaScript, NaN is a number that is not a legal number.

    Infinity:

  • A number reaches Infinity when it exceeds the upper limit for a number:.
  • 1.797693134862315E+308.
  • Infinity is a number that represents positive infinity.
  • -Infinity represents negative infinity.

    • Example for infinity:

    let x = 1.797693134862315E+308;
    let y = x * 1.001;
    console.log(y);
    output: infinity

    Null:

  • A value that is explicitly specified by the keyword “null” is known as null value
  • Null is of type object in Javascript

    • Example for null:

    var test=null;
    console.log(test);
    //null

    Undefined:

  • Undefined is of type undefined.

    • Example for undefined:

    var testNum;
    console.log(typeof testNum);
    //Undefined: Here testNum is not defined

5.)‘this’ keyword in OOPs in JavaScript :

  • In Javascript this keyword refers to an object.
  • In an object method, this refers to the object.
  • Alone, this refers to the global object.
  • In a function, this refers to the global object.
  • In a function, in strict mode, this is undefined.
  • In an event, this refers to the element that received the event.

6.i)DOM and Document in JavaScript :

    Document Object Model

  • When a web page is loaded, the browser creates a Document Object Model of the page.
  • The HTML DOM model is constructed as a tree of Objects.
  • With the object model, JavaScript gets all the power it needs to create dynamic HTML .
  • JavaScript can change all the HTML elements,attributes, css styles in the page .
  • JavaScript can react to all existing HTML events in the page
  • JavaScript can create new HTML events in the page

    HTML Dom Document

  • The HTML DOM document object is the owner of all other objects in your web page.
  • The document object represents your web page.
  • If you want to access any element in an HTML page, you always start with accessing the document object..

    • Examples to use dom object to access and manipulate

    Few Methods for finding html elements:

  • document.getElementById(id)
  • document.getElementsByTagName(name)
  • document.getElementsByClassName(name)

    • Few Methods for adding and deleting html elements:

  • document.createElement(element)
  • document.removeChild(element)
  • document.appendChild(element)
  • document.write(text)

6.ii)Array methods in JavaScript :

    Array methods

  • toString() converts an array to a string of (comma separated) array values.
  • The join() method also joins all array elements into a string.It behaves just like toString(), but in addition you can specify the separator:
  • The pop() method removes the last element from an array.The pop() method returns the value that was "popped out"
  • The push() method adds a new element to an array (at the end).The push() method returns the new array length
  • The shift() method removes the first array element and "shifts" all other elements to a lower index.The shift() method returns the value that was "shifted out"
  • The unshift() method adds a new element to an array (at the beginning), and "unshifts" older elements.The unshift() method returns the new array length.
  • The length property provides an easy way to append a new element to an array:
  • Array elements can be deleted using the JavaScript operator delete.Using delete leaves undefined holes in the array.Use pop() or shift() instead.
  • The concat() method creates a new array by merging (concatenating) existing arrays.The concat() method does not change the existing arrays. It always returns a new array.
  • The splice() method adds new items to an array.With clever parameter setting, you can use splice() to remove elements without leaving "holes" in the array
  • The slice() method slices out a piece of an array into a new array.
